# Ocular Medicine — Clinical Reference Knowledge

## Pupillary Response Assessment
- **Normal pupil size**: 2–5mm (bright light), 4–8mm (dim light)
- **PERRLA**: Pupils Equal, Round, Reactive to Light and Accommodation
- **Consensual response**: Light in one eye should constrict both pupils
- **RAPD (Marcus Gunn)**: Afferent pupillary defect → optic nerve/retinal disease. Swinging flashlight test

## Pupil Size Abnormalities
| Finding | Unilateral | Bilateral |
|---------|-----------|-----------|
| Miosis (<2mm) | Horner's syndrome, iritis | Opioids, pontine lesion, pilocarpine |
| Mydriasis (>8mm) | CN III palsy, trauma | Sympathomimetics, anticholinergics, brain death |
| Anisocoria (>1mm diff) | Physiological (20%), CN III palsy, Horner's | Bilateral pathology |

## Intraocular Pressure (IOP)
- **Normal**: 10–21 mmHg
- **Ocular hypertension**: >21 mmHg without optic nerve damage
- **Glaucoma suspect**: IOP >21 mmHg + suspicious disc/field changes
- **Acute angle-closure**: IOP >40 mmHg + pain + halos + nausea → emergency

## Fundoscopy Findings
- **Papilledema**: Bilateral disc swelling → raised intracranial pressure. Emergency if progressive
- **Cotton wool spots**: Retinal nerve fiber layer infarcts → HTN, DM, HIV, vasculitis
- **Dot/blot hemorrhages**: Deep retinal hemorrhages → diabetic retinopathy, venous occlusion
- **Flame hemorrhages**: Superficial nerve fiber layer → hypertensive retinopathy
- **Roth spots**: White-centered hemorrhages → endocarditis, leukemia, severe anemia
- **Neovascularization**: New vessel growth → proliferative diabetic retinopathy, CRVO

## Visual Acuity Reference
- **6/6 (20/20)**: Normal acuity
- **6/12**: Minimum for driving (many jurisdictions)
- **6/60**: Legal blindness threshold (WHO)
- **Counting fingers / Hand movements / Light perception**: Severe visual impairment grading

## Eye Movement & Cranial Nerves
- **CN III (Oculomotor)**: Medial, superior, inferior rectus + inferior oblique. Ptosis + mydriasis if palsy
- **CN IV (Trochlear)**: Superior oblique. Head tilt test for diagnosis
- **CN VI (Abducens)**: Lateral rectus. Most common CN palsy → raised ICP

## Red Flags Requiring Immediate Escalation
- Sudden unilateral vision loss → retinal artery occlusion, optic neuritis
- Fixed dilated pupil + headache → CN III compression, herniation
- Acute IOP >40 mmHg → angle-closure glaucoma emergency
- New papilledema → urgent imaging to rule out intracranial mass
- Bilateral vision loss → cortical blindness, bilateral optic neuritis
